## How to use this project
Replace my key in this code with your key:
~~~sh
@GET("v3/latest?apikey=YOUR_API_KEY_HERE&currencies=EUR&base_currency=VND")
~~~
You can get the key in [here](https://app.currencyapi.com/api-keys).
